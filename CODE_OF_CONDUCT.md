# Código de Conduta

## Nossa Promessa

Nós, como membros, colaboradores e líderes, nos comprometemos a tornar a participação em nossa comunidade uma experiência livre de assédio para todos, independentemente de idade, aparência física, deficiência, etnia, identidade ou expressão de gênero, nível de experiência, nacionalidade, religião ou orientação sexual.

## Nossos Padrões

Exemplos de comportamentos que contribuem para um ambiente positivo incluem:

- Usar linguagem acolhedora e inclusiva;
- Respeitar diferentes pontos de vista e experiências;
- Aceitar críticas construtivas;
- Demonstrar empatia com outros membros;
- Colaborar de forma ética e respeitosa.

Exemplos de comportamentos inaceitáveis incluem:

- Linguagem ofensiva ou discriminatória;
- Ataques pessoais ou políticos;
- Assédio público ou privado;
- Compartilhamento de informações privadas sem permissão;
- Qualquer conduta inadequada em ambiente profissional.

## Responsabilidades de Aplicação

Os responsáveis pelo projeto têm a responsabilidade de esclarecer e aplicar os padrões de comportamento aceitável.

Eles podem remover, editar ou rejeitar contribuições que não estejam alinhadas com este Código de Conduta.

## Escopo

Este Código de Conduta se aplica a todos os espaços do projeto, incluindo repositórios, issues, pull requests e canais oficiais.

Também se aplica quando alguém representa o projeto publicamente.

## Aplicação

Casos de comportamento inadequado podem ser reportados aos responsáveis pelo projeto.

Todas as denúncias serão analisadas de forma justa e confidencial.

## Diretrizes de Aplicação

As consequências para violações incluem:

1. Correção
2. Advertência
3. Suspensão temporária
4. Banimento permanente

## Atribuição

Este Código de Conduta é adaptado do Contributor Covenant.
